Feb 21, 2017 | Hits

MVP　设计模式.

Model

主要负责数据交互.
实现数据请求方法，使用回调接口.

View

定义接口. 操作各种控件.

Presenter

负责Model和View的交互. 持有Model和View的引用, 调用Model中的接口，在回调中调用View中的方法

Activity

实现 View 接口
持有 Presenter　的引用
